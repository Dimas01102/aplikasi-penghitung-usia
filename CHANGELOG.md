# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-07

### Added
- Initial release of Aplikasi Penghitung Usia
- GUI application using Tkinter
- Age calculation feature (Years, Months, Days)
- Input validation (date format, leap year, future date)
- Additional features:
  - Total days lived
  - Days countdown to next birthday
  - Zodiac sign determination
- Modern and user-friendly interface
- Error handling with message boxes
- Reset functionality
- Documentation:
  - Complete README
  - Function and module documentation
  - Presentation guide
  - Flowchart generation scripts

### Features
- Cross-platform compatibility (Windows, macOS, Linux)
- No external dependencies (all built-in Python libraries)
- OOP implementation with clean code structure
- Comprehensive input validation
- Professional UI design

### Documentation
- Setup script for system verification
- Flowchart generation for algorithm visualization
- Detailed function and module documentation
- Complete presentation guide with Q&A scenarios
